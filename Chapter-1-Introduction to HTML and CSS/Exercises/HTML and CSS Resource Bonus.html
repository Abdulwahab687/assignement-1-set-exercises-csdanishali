HTML & CSS Quick Reference Guide

A handy guide for web development beginners - created by a fellow student who's been there!

HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages. Here are some of the most common tags we use:

Headings: <h1> to <h6>

Headings help organize your content. <h1> is the most important (usually the page title) and <h6> is the least important.

<h1>This is a Main Heading</h1>
<h2>This is a Section Heading</h2>
<h3>This is a Subsection Heading</h3>

Paragraphs: <p>

Paragraphs are used for blocks of text. Browsers automatically add some space before and after each paragraph.

<p>This is a paragraph of text. Paragraphs are block-level elements.</p>
<p>This is another paragraph.</p>

Links: <a>

The anchor tag is used to create hyperlinks to other web pages, files, or locations within the same page.

<a href="https://www.example.com">Visit Example Website</a>
<a href="#section-id">Jump to Section</a>

Images: <img>

The image tag embeds images into your webpage. It's a self-closing tag (no end tag needed).

<img src="image.jpg" alt="Description of image">

Lists: <ul>, <ol>, <li>

Lists help organize information. Use <ul> for unordered (bullet) lists and <ol> for ordered (numbered) lists.

<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
</ol>

Div & Span: <div> & <span>

<div> is a block-level container, while <span> is an inline container for text and elements.

<div>This is a block-level container</div>
<p>This paragraph contains a <span>highlighted</span> word.</p>

Code & Pre: <code> & <pre>

Use <code> for inline code and <pre> to preserve formatting of code blocks.

<p>The <code>console.log()</code> function outputs to the console.</p>

<pre><code>function greet() {
  console.log("Hello world!");
}</code></pre>

Tables: <table>, <tr>, <td>

Tables are used to display data in rows and columns.

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Row 1, Cell 1</td>
    <td>Row 1, Cell 2</td>
  </tr>
</table>

Forms: <form>, <input>, <button>

Forms allow users to input data and submit it.

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">
  
  <button type="submit">Submit</button>
</form>

Comments: <!-- -->

Comments aren't displayed on the page but help document your code.

<!-- This is a comment. It won't be visible on the page. -->

CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. Here are some key properties:

Color Properties

Control the color of text and backgrounds.

/* Text color */
color: blue;
color: #FF5733;
color: rgb(255, 87, 51);

/* Background color */
background-color: yellow;
background-color: #F2F2F2;
background-color: rgba(242, 242, 242, 0.5); /* with transparency */

Font Properties

Control the appearance of text.

/* Font family */
font-family: Arial, sans-serif;

/* Font size */
font-size: 16px;
font-size: 1.2em;
font-size: 120%;

/* Font weight */
font-weight: bold;
font-weight: 700;

/* Font style */
font-style: italic;

Box Model Properties

Control the spacing and borders around elements.

/* Margin - space outside the element */
margin: 10px;
margin: 10px 20px; /* top/bottom left/right */
margin: 5px 10px 15px 20px; /* top right bottom left */

/* Padding - space inside the element */
padding: 10px;

/* Border */
border: 1px solid black;
border-radius: 5px; /* rounded corners */

Layout Properties

Control how elements are positioned and displayed.

/* Display types */
display: block;
display: inline;
display: flex;
display: grid;

/* Position */
position: relative;
position: absolute;
position: fixed;

Flexbox

A modern layout model for arranging items in rows or columns.

/* Parent container */
display: flex;
flex-direction: row; /* or column */
justify-content: space-between;
align-items: center;

/* Child items */
flex: 1;
align-self: flex-start;

Text Properties

Control text alignment and decoration.

/* Text alignment */
text-align: center;
text-align: left;
text-align: right;

/* Text decoration */
text-decoration: underline;
text-decoration: none; /* remove underlines from links */

Transform & Transition

Add animations and transformations to elements.

/* Transforms */
transform: rotate(45deg);
transform: scale(1.5);

/* Transitions */
transition: all 0.3s ease;
transition: background-color 0.5s, transform 0.3s;

Media Queries

Create responsive designs that adapt to different screen sizes.

/* Apply styles only on screens smaller than 600px */
@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
  .container {
    width: 100%;
  }
}
