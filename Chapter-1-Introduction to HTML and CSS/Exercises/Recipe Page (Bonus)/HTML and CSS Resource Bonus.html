Introduction

This page highlights common HTML elements and CSS properties using clear examples.

HTML Tags

<html>

Defines the root of the HTML document.

<html lang="en">...</html>

<head>

Contains metadata and links.

<head>
    <meta charset="UTF-8">
    <title>Page</title>
</head>

<body>

Holds all the visible content.

<body>
    <h1>Welcome</h1>
    <p>Content</p>
</body>

CSS Properties

Text Styling

font-family: Arial;
font-size: 16px;
text-align: center;

Box Model

width: 100%;
padding: 20px;
margin: 10px;
border: 1px solid;

Flexbox

display: flex;
justify-content: center;

Best Practices

HTML Tips

  • Use semantic tags for structure
  • Always include alt text for images
  • Keep your code clean and readable

CSS Tips

  • Use external stylesheets
  • Use class names that describe purpose
  • Make your layout responsive
