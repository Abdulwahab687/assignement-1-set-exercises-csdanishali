Basic HTML Tags

Here’s a quick look at the most useful tags I’ve been using:

Tag Description Example
<h1>–<h6> Headers, from big to small. <h2>Intro</h2>
<p> Paragraph text. <p>Hello world</p>
<a> Creates a link. <a href="page.html">Go</a>
<img> Displays an image. <img src="cat.jpg" alt="cat">
<ul>, <ol>, <li> Lists: unordered, ordered, list item. 
<ul>
  <li>Item</li>
</ul>
<div> Box to group stuff. <div class="section">...</div>
<span> Inline stylable text. <span style="color:green">Green!</span>
<br> Line break. Hello<br>World
<hr> Horizontal divider. <hr>
<input> Form input field. <input type="text">

CSS Styling Examples
