Web Development Reference Guide

Your comprehensive HTML & CSS companion

Overview

A handy reference for common HTML elements and CSS properties with practical examples.

HTML Elements

<html>

Document root element

<html lang="en">...</html>

<head>

Document metadata container

<head>
    <meta charset="UTF-8">
    <title>My Website</title>
</head>

<body>

Document content container

<body>
    <h1>Hello World</h1>
    <p>This is my website.</p>
</body>

Heading Elements

Six levels of headings

<h1>Primary Heading</h1>
<h2>Secondary Heading</h2>

<p>

Text paragraph

<p>This is a paragraph of text.</p>

<a>

Hyperlink element

<a href="https://example.com">Visit Example</a>

<img>

Image element

<img src="photo.jpg" alt="Description of image">

List Elements

Ordered and unordered lists

<ul>
    <li>List item one</li>
</ul>

<div> and <span>

Generic containers

<div>Block-level container</div>
<span>Inline container</span>

Table Elements

Structured data display

<table>
    <tr>
        <th>Header cell</th>
        <td>Data cell</td>
    </tr>
</table>

Form Elements

User input collection

<form>
    <input type="text" placeholder="Enter text">
    <button type="submit">Submit</button>
</form>

<pre> and <code>

Code formatting elements

<pre><code>function example() { return true; }</code></pre>

Semantic Elements

Descriptive structural elements

<header>Page header content</header>
<main>Main page content</main>

CSS Properties

Color Properties

color: #2a2a2a;
background-color: #f8f8f8;

Text Styling

font-family: 'Helvetica', sans-serif;
font-size: 18px;
text-align: left;

Box Model

width: 80%;
padding: 15px;
margin: 20px auto;
border: 2px solid #ddd;

Layout Properties

display: block;
position: absolute;

Flexbox Layout

.flex-container {
    display: flex;
    justify-content: space-between;
}

Grid Layout

.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
}

Animation Properties

transition: opacity 0.5s ease;
animation: fadeIn 2s;

Media Queries

@media screen and (max-width: 600px) {
    body { font-size: 16px; }
}

Best Practices

HTML Best Practices

  • Use semantic HTML elements when appropriate
  • Always include alternative text for images
  • Maintain a logical document structure

CSS Best Practices

  • Keep styles in separate CSS files
  • Use descriptive class and ID names
  • Implement mobile-first responsive design
